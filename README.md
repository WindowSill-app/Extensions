# WindowSill Extensions

Official extensions for [WindowSill](https://getwindowsill.app/), the modern Windows productivity toolbar.

## About

This repository contains first-party extensions developed and maintained by the author of WindowSill and the community. Each extension adds specialized functionality to enhance your Windows workflow.

## Extensions

| Extension | Description |
|-----------|-------------|
| **App Launcher** | Quick-launch applications and organize them into groups. Supports UWP apps, shortcuts, executables, and folders. |
| **Clipboard History** | Access your clipboard history with support for text, images, files, URLs, colors, RTF, HTML, and more. |
| **DevToys** | Developer utilities including JSON/XML formatting, Base64 encoding/decoding, text escaping, and data conversion tools. |
| **Image Helper** | Resize, compress, and convert images between formats directly from the toolbar. |
| **Media Control** | Control media playback across applications—play, pause, skip tracks without leaving your current window. |
| **Performance Counter** | Monitor CPU, memory, and GPU usage in real-time from your toolbar. |
| **Short-Term Reminder** | Set quick reminders with full-screen notifications to stay on track. |
| **Teams** | Control Microsoft Teams meetings—toggle mute, camera, and leave calls directly from the toolbar. |
| **Unit Converter** | Convert between temperature units, dimensions, and currencies with live exchange rates. |
| **URL Helper** | Generate QR codes and shorten URLs quickly. |
| **Web Browser** | Quick-launch your preferred web browser (Chrome, Edge, Firefox, Brave, Opera, Vivaldi, Zen). |
